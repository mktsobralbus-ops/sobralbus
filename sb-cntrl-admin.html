Publicar no Site

Gera as páginas HTML e envia para o site automaticamente

Como funciona: ao clicar em publicar, o sistema busca todos os veículos ativos no banco, gera os arquivos HTML de cada um e faz o commit direto no GitHub. O Vercel detecta a mudança e publica o site automaticamente em 1-2 minutos.

Instruções

Como usar o painel

1. Adicionar veículo
Clique em "Adicionar Veículo", preencha os dados e clique em "Salvar". O veículo já fica salvo no banco de dados.
2. Editar ou remover
Na lista de Veículos, use os botões "Editar" ou "Remover" em cada linha.
3. Publicar no site
Depois de fazer suas alterações, vá em "Publicar no Site" e clique no botão. Isso gera as páginas e atualiza o site automaticamente — sem precisar editar nenhum arquivo manualmente.
Sobre este painel
Este arquivo não está linkado em nenhuma parte do site. Acesse digitando a URL diretamente. Não compartilhe o link.
Sobre o Token do GitHub
Na primeira vez que clicar em "Publicar", o painel vai pedir seu GitHub Token. Ele fica salvo só neste navegador (nunca no código). Se quiser trocar ou remover, use o botão abaixo.
