Instruções de Uso

Como adicionar um novo veículo ao site

Fluxo completo
Passo 1 — Adicionar as fotos
Coloque as fotos na pasta correta:
• Rodoviário → assets/rodoviario/NOME-DA-PASTA/
• Urbano → assets/urbano/NOME-DA-PASTA/
• Micro → assets/micro/NOME-DA-PASTA/
Nomeie as fotos: A.png, B.png, C.png...
Passo 2 — Preencher o formulário
Vá em "Adicionar Veículo" neste painel, preencha todos os campos e clique em "Gerar Código".
Passo 3 — Colar no onibus.js
Copie o código gerado e cole dentro do array catalogo no arquivo onibus.js, antes do comentário // ── ADICIONE NOVOS VEÍCULOS AQUI ──
Passo 4 — Rodar o gerador
No terminal, dentro da pasta do projeto:
node gerar-paginas.js
Isso gera automaticamente o HTML da página do veículo e atualiza o catalogo.js.
Passo 5 — Deploy
Faça o commit e push para o repositório Git. O Vercel detecta automaticamente e publica a nova versão.
git add .
git commit -m "Adiciona [nome do veículo]"
git push
Sobre este painel
Este arquivo admin.html não está linkado em nenhuma parte do site. Para acessá-lo, você digita a URL diretamente. Por segurança adicional, renomeie o arquivo para algo não óbvio — ex: sb-ctrl-2024.html — e nunca o exponha publicamente.
